HOW TO FIX "File Not Found" ERROR
=================================

Error in logs: "bash: deploy.sh: No such file or directory"
Means: Files weren't deployed to Azure

Solution:

1. Make sure files are in GitHub:
   - Go to: https://github.com/Amanraj9910/SLD
   - Look for: app.py, requirements.txt, .deployment
   - If missing: Upload using GitHub Web interface

2. Change Startup Command:
   - Azure Portal → Configuration
   - Startup Command: python app.py
   - Click Save

3. Restart App Service

4. Check logs in 2-3 minutes

That's all you need!
