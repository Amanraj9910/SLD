   c) Select branch: main

TROUBLESHOOTING:

If still showing PM2:
1. In Azure Portal → Configuration → General Settings
2. Make sure Runtime Stack is set to Python 3.11 (NOT Node.js)
3. Make sure Startup Command is: bash deploy.sh
4. Delete package.json from root if it exists
5. Restart App Service

If Port Error (8080 vs 8000):
1. App Service sets PORT environment variable
2. Python app reads from PORT env var
3. If not working, set in Portal → Application Settings → PORT=8000

If Import Errors:
1. Check Log stream for detailed error
2. Verify app.py, main.py exist at correct paths
3. Ensure all dependencies in requirements.txt
4. Check disk quota in Kudu console (https://YOUR-APP.scm.azurewebsites.net/api/dump)

If 502 Bad Gateway:
1. App might be crashing on startup
2. Check Application Insights for errors
3. Verify local paths match deployment paths
4. Run: az webapp log tail --resource-group RG_NAME --name APP_NAME
